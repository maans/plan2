# Upload til GitHub

1. Pak ZIP-filen ud.
2. Upload **hele indholdet** til roden af repositoryet `maans/Plan`, så eksisterende filer overskrives.
3. Bevar mapperne. Især skal disse filer ende på GitHub:
   - `index.html`
   - `catalog.json`
   - `plan-library/catalog.json`
   - `andre/camp-26/forloeb.json`
   - `andre-9/praktiske-kreative-fag/camp-26/forloeb.json` (kompatibilitet)
4. Vent på GitHub Pages-opdateringen og genindlæs Plan.

Camp 26 skal derefter stå under **Andre** uden klassetrin. Knappen **Se og brug** står direkte under beskrivelsen.
