# Ændringslog

## Plan v1.2.0 (Stable) — 10. juli 2026

### Kalender
- Filter og Hent har byttet plads, så Filter er direkte tilgængeligt i kalenderens topbjælke.
- Kalenderfiler registreres som kilder med stabil intern identitet.
- Opdaterede ICS-filer genkendes via UID og RECURRENCE-ID frem for kun filnavn.
- Ved import af én ICS-fil i en åben kalender vælges tydeligt mellem at flette med den åbne kalender, vise den nye kalender i samme widget, åbne den som en ny kalender eller annullere.
- Ved import af flere ICS-filer bevares valget mellem flere kalendere og én samlet kalender.
- Eksisterende dag-, uge-, måned- og listevisninger samt kalenderens øvrige menuer er bevaret.

### Bibliotek og pakker
- Indholdskontroller, pakkekontroller og katalogstørrelser er genberegnet.
- Alle pakker er valideret mod kataloger, manifests og filstruktur.
- 17 løse relationer til manglende linkdatasæt i Fællessamlinger 2026–27 er fjernet, mens de synlige links er bevaret.

### Kompatibilitet
- Eksisterende Plan- og Tavle-dataformat bevares.
- Gamle KaPardy-spil og eksisterende bibliotekspakker bevares.
