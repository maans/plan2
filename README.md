# Plan 2.0

**Plan er lærerens arbejdsplads gennem skoleåret.** Her samles forløb, undervisningsgange, tavler, materialer og data, så arbejdet kan fortsættes og genbruges.

## Fire niveauer

**Undervisningsforløb → Sessioner → Tavler → Widgets**

Forløbet er den store ramme. Sessionerne deler arbejdet op. Tavlerne samler det, der skal bruges, og widgets er indholdet på tavlen.

## Planoversigt

Planoversigt er den fælles indgang til:

- **Mit arbejde:** Forløb, Bibliotek og Sessioner
- **Elever og fælles grundlag:** Fælles data, Datakilder og Sammenhænge
- **Materialer og data:** Sessionsdata, Bearbejd data og Medier
- **Backup og sikkerhed:** Gendan samt Lager og sikkerhed

Bibliotek leverer indhold, som hentes til en lokal kopi. Sammenhænge viser, hvor et datasæt bruges. Gendan og Lager og sikkerhed hjælper med at beskytte arbejdet.

## Gem og flyt arbejdet

Plan gemmer lokalt på den enhed, der bruges. Tag derfor jævnligt sikkerhedskopier, især før større oprydning eller flytning til en anden enhed.

## Separate filer

`index.html` indeholder appen. Håndbog, screenshots og øvrige materialer ligger som separate filer og åbnes via relative stier, kataloger eller webadresser. De indlejres ikke som data i appens HTML.

PDF-håndbogen ligger i `manualer`, og `manualer/catalog.json` angiver den lokale fil og en webadresse som reserve.

## Hjælp

Hjælpen i Plan giver korte svar midt i arbejdet. **Plan – Lærerhåndbog** er det visuelle opslagsværk og kan åbnes fra Hjælp.
