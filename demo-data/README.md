# Haubro Efterskole – komplet demonstrationsdata 2026/27

Pakken indeholder 150 fiktive elever med portrætter og stabile relationer via `Unilogin`. Danskhold, niveaudeling, skolekalender, tre fagsamtaler og karakterudvikling er synkroniseret med V5-stamdata.

Alle personer, kontaktoplysninger, besvarelser, karakterer og aktiviteter er fiktive.
