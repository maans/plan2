# Plan · skabeloner

Skabeloner er tomme, men ikke blanke. De indeholder tavler, tomme widgets og en tydelig arbejdsgang, så læreren kan komme i gang uden at starte fra en helt tom tavle.

Skabelonerne indeholder ikke elevdata og ikke indlejrede materialer. Tilføj selv materialer, datoer, elevdata og noter i din lokale kopi af Plan.
