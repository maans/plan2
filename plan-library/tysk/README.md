# Tysk

Årsplaner og forløb om tysk sprog, kultur, litteratur og kommunikation.

Hvert udgivet årsforløb ligger i sin egen undermappe med mindst:

- `forloeb.json` – den kontrollerede Plan-årsforløbspakke
- `README.md` – en kort beskrivelse af indhold, klassetrin og eventuelle materialer
- eventuelt `kalender.ics` og `forside.webp`

Forløbet skal samtidig være registreret i `../catalog.json`.
