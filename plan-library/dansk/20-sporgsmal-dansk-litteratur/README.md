# 20 spørgsmål fra professoren · Dansk litteratur

Professorforløb v2 med kilder, medier, Huskekortmatrix og KaPardy.

## Indhold

- 10 tavler
- Kilde- og medieliste med linkwidgets
- Fire faglige kategoritavler
- Huskekortmatrix koblet til den samme KaPardy-spørgsmålsbank
- Professorens prøveværksted
- Det eksisterende 20-spørgsmåls KaPardy-spil som afslutning

## Kildeprincip

Kilderne er synlige på tavlen og kan åbnes som links eller webvisning. Eleverne skal bruge dem til at forklare svarene, ikke bare huske facit.
