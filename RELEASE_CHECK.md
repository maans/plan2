# Releasekontrol – Plan v1.2.0

Dato: 10. juli 2026

## Gennemført

- `index.html` er baseret på den fungerende originale Plan-app med de seneste konsoliderede kalenderændringer.
- JavaScript-syntaks er kontrolleret for alle 113 indlejrede scriptblokke: 0 syntaksfejl.
- Bibliotekets pakke- og katalogmetadata stammer fra den opryddede og genvaliderede repository-udgave.
- 17 løse relationer i Fællessamlinger 2026–27 er fjernet uden at fjerne de synlige links.
- Version, buildoplysninger, README og ændringslog er opdateret til Plan v1.2.0 (Stable).

## Kalenderændringer i denne release

- Filter og Hent har byttet plads.
- Kalenderkilder spores stabilt.
- ICS-opdateringer genkendes via UID og RECURRENCE-ID.
- Én ny ICS-fil giver valg mellem flet, vis her, åbn som ny eller annuller.
- Flere ICS-filer beholder valget mellem flere kalendere og én samlet kalender.
- Eksisterende dag-, uge-, måned- og listevisninger er bevaret.

## Bemærkning

Kontrollen her er en statisk kode- og pakkekontrol. Den endelige betjening på iPad, Quine, Safari og desktop bør stadig prøves i den faktiske app efter upload.
